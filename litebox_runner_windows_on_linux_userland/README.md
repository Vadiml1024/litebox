# litebox_runner_windows_on_linux_userland

A CLI runner for executing Windows PE binaries on Linux using LiteBox.

## Overview

This runner combines `litebox_shim_windows` and `litebox_platform_linux_for_windows` to enable running Windows programs on Linux with optional API tracing.

## Usage

```bash
litebox_runner_windows_on_linux_userland <windows_binary.exe>
```

## Status

This is part of Phase 1 implementation. Currently in development.
